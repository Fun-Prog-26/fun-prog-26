# 3. Starter Code

The code you accept as the 'starter' is  a stack project with the addition of 
 - starter code that helps you to get started including reading in a csv file and building up an instance of a Lecturer datatype. 
 - a  csv file containing details of  lecturers

    - two records (two lecturers)
    - five   fields 

    | Field Name  |Field Type  | Validation Rule | 
    |:-------------:|:-------------:|:-------------|
    | lectureID  | String | First Char is aphlabetic, then 3 digits  | 
    |name  |  String | length <= 15  |
    | email  | String  | must have a '@' and a '.' later in list |
    | departmentID  | String  |none |
    | availablHours  | int  |  availableHours should be between 0 and 22, inclusive |
