# StockQuotes Case Study

Details of Case Study

## Aim

This case study should act as a basis for your Programming Assignment 2. This case studies is based on '*Haskell in Depth, Vitaly Bragilevsky, MEAP , Manning Leanpub* ' - Chapter 3.

## Specification

This program 
 - reads in stock data from a csv file (quotes.csv)
 - interprets the data (using cassava package)
 - allows the user to specify which statistics to report and in which format. The available formats are 
    - text based statistics
    - html report on statistics
 - Using functions, calculate various value based on the data, e.g. minimum opening values, maximum opening values, average etc.
